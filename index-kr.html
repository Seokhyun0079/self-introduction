프로젝트

GIF Cutter (클릭 시 GitHub로 이동)

React TypeScript Vite gifuct-js

GIF 파일의 프레임을 개별적으로 표시하고, 특정 프레임을 이미지로 추출할 수 있는 웹 애플리케이션입니다. 프레임 누적 처리에 의한 정확한 GIF 표시, 필름 스트립 UI, 프레임 추출 기능을 구현했습니다. 기술적으로는 GIF의 압축 방식에 대응한 정확한 프레임 합성, disposalType과 transparentIndex의 적절한 처리, 실시간 프리뷰 기능을 특징으로 하고 있습니다.

Flipboard (클릭 시 GitHub로 이동)

Angular TypeScript RxJS Express

전통적인 플립보드(Flip Board)를 웹으로 구현한 애플리케이션입니다. 공항이나 기차역에서 사용되던 기계식 전시판의 독특한 시각적 효과와 애니메이션을 웹에서 재현하여, 사용자가 입력한 텍스트가 실제 플립보드처럼 각 글자가 회전하며 표시됩니다.
