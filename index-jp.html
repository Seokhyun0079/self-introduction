プロジェクト

Flipboard (クリックでGitHubへ)

Angular TypeScript RxJS Express

伝統的なフリップボード（Flip Board）をウェブで実装したアプリケーションです。 空港や駅で使用されていた機械式の表示板の独特な視覚効果とアニメーションをウェブで再現し、 ユーザーが入力したテキストが実際のフリップボードのように各文字が回転しながら表示されます。

Jumplix (クリックでGitHubへ)

JavaScript Chrome Extension

Netflixの視聴体験を向上させるChrome拡張機能です。シリーズのイントロを自動的にスキップし、 現在のエピソードが終わると自動的に次のエピソードに進む機能を提供しています。

GitHub Blog Editor (クリックでGitHubへ)

React Node.js Express Octokit

GitHubをブログとして活用できるエディタープロジェクトです。投稿作成、ファイル添付、JSON形式での保存などの機能を提供しています。 フロントエンドはReact、バックエンドはNode.jsとExpressを使用し、GitHub APIとの連携にはOctokitを活用しています。

Tree Table ID Generator (クリックでGitHubへ)

Go Next.js MySQL

階層構造データの自動ID生成システムを開発しました。親子関係を維持しながら IDを自動生成し、データ構造を可視化する機能を実装しました。

LLM 学習プロジェクト (クリックでGitHubへ)

Python LLM

「作りながら学ぶ！LLM自作入門」を通じて、大規模言語モデルの基礎から実装までを学習中です。 理論的な理解と実践的なコーディングを通じて、AI技術の本質的な理解を深めています。

JavaScript Tetris (クリックでGitHubへ)

JavaScript HTML CSS

JavaScriptを使用して、クラシックなテトリスゲームを実装しました。 オブジェクト指向プログラミングの概念を活用し、ゲームロジックとUIを分離して設計しました。

NoComment (クリックでGitHubへ)

React Node.js MongoDB

悪意のあるコメントがなくなることを願い、 テキストベースのコメントシステムの代わりにマウスで絵を描く方式を採用した個人用ホームページです。 フロントエンドはReact、バックエンドはNode.js、データベースはMongoDBを使用しています。
